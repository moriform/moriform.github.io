这个月买了什么 十一月算是购物大月，我虽然不在国内没有办法过双十一，但是还是一贯在黑五买了些东西。 Launchpad Pro mk3 一个19年的老玩意儿了。16年的时候买了Launchpad mk2的版本打了几个灯光效果很酷炫的工程之后就一直用于DrumPad弹和弦，以及和Ableton live一起使用在直播的时候作为通道条的控制器。 在Ebay上面120镑买的几乎全新的mk3，价格和当时闲鱼上面的最低价差不多了。新的版本多了版面的自定义功能可以使用不同的预设。然后多了十分好用的和弦功能和音阶选择功能，很适合没有灵感的时候随机漫游。 以及Pro版本有专属的音序器功能，暂时没有时间去摸索，估计是可以和其他硬件合成器搭配使用的方案。 Mpowersynth 红塔山(Melda production)的波表合成器打骨折，加上折扣券28镑收了，算是逐渐集齐红塔山全家桶的又一步。 特点是极低的cpu消耗以及红塔山的特色效果器路由和无限可能性。 当然还有红塔山特色的让人不想靠近的UI。 Rond 128元 暄穗之前一直使用Life cycle来进行日常的追踪和记录，在使用的过程中发现了其局限性：其一是太长时间没有更新，像是亖了一样。其二是地图如果有所偏移的话修改方式极其诡异甚至无法修改。 所以在少数派搜索Life cycle的时候看见了Rond这款软件。给我的整体感觉就是一个持续更新的Life Cycle，而且还是可以买断的，所以试用了三天就买了。给人感觉相当不错。 同时下载了一个叫做一生足迹的APP一直放在手机后台记录。要说两者的区别的话Rond和Life cycle都是记录事件，记录在一个地方呆的时间有多长。而一生足迹是记录具体地图上经历过的路径。 ELEGOO UNO R3 用于处理学校的Creative Coding for Sound大作业。但是Arduino的价格太高所以选择了替代主板。同时购买的还有ECG贴片，AD8232组件和一盆绿植。大概也能猜出想要做什么东西了。 游戏 亚路塔 关于见习勇者打工打怪建设文明乡镇的故事。一个迷你且完善的类潜水员戴夫的经营动作类游戏。 像素人物很可爱，但是刷钱的过程有些肝弥补了游戏时长短的缺点。估计20小时以内可以玩完。 星之终途 既然是Key社的游戏还是短篇还是4折促销就不得不买了。

Max治好了我的巴托比症候群

这是什么 上周暄穗花了几天的时间完成了一个Max工程作为创意声音编程的项目提交了上去。这个项目改变了暄穗对于创作的底层思考。 大纲： 简述这个项目是什么 Lofi音乐机 随机性 为什么说我有巴托比 对于创作的胆怯 对于年龄和是不是应该做的事情，拖延症让我对于创作持续延后；acfun截的僧人的图 对于作画便是每个像素的考量，对于音乐便是每一帧的计较，而不是整体pattern和texture的思想 怎么治好的 通过pattern来管理整个作品， 通过并无答案，只要原理正确即使是随机也能够有很好的效果，更何况是自己管理的呢 关于项目 稍微说一下项目本身。 这是一个自称“自动Lofi生成系统”的MIDI生成片段。需要和任意一个daw连接，通过MIDI演奏乐器达到完成的成品。 如果你玩过《节奏医生》就知道里面2-4关卡海之歌（就是让你吃妮可和科尔狗粮的那一关）只需要不停地按下空格键就可以和着背景的爵士乐演奏一个听上去似乎还挺和谐的旋律。曾经我在这一关弹了10多分钟也不知道如何完成关卡，但也就是这十分钟给了我关于音乐新的感觉：1.音乐是可以无限延申的。2.有一种算法可以让随机的旋律和背景很好地融合在一起。如果乐理知识再深一点就知道这个算法的原理就是让旋律弹奏背景和弦的不同组成音。 于是，这两个思想就构成了课程项目的作业。暄穗按照其对于Lofi音乐拙劣理解和代码块的不成熟使用在几天时间里搭建了这个系统：背景音就是钢琴和弦，Lofi鼓就是最基础的那几个鼓组，贝斯就是和弦根音，旋律就是和弦随机音。贝斯和旋律音的时长完全从二分音符到十六分音符随机选择。这样一个简单Lofi音乐生成系统就搭建好了，没有使用更多的织体和各种和声写作方法，仅仅就是拼贴。唯一音乐知识的补足是上网搜索了几个Lofi合适的和弦进行，发现只要是同一音阶的七和弦拼上都挺不错。 在这个系统中自动生成的音乐充满了随机性。和弦是随机的，随机和和弦进行，以及随机的转调处理。鼓组是随机的，4个基本鼓组随机选择。贝斯是随机的，音符时长以及是否高八度处理是随机的。旋律是随机的，甚至对于音符选择和弦中的音还是音阶中和弦内的音也有权重。 于是，基于规则内的随机，显示出了超乎想象的效果，即出乎意料地正常。如果把这个系统放在直播的时候作为背景音乐运行个半个小时估计也没有人在意其中的微妙感。 巴托比症候群 巴托比出自梅尔维尔的小说《书记员巴托比》。后比拉·马塔斯以此为名写了一本《巴托比症候群》提出了这样一个现象：关于作家突然拒绝创作。 为何拒绝创作？书里面给出了各种各样的原因。但是，我为什么不创作？我似乎得了这样一种空想症，以至于我在很久之间都没有继续进行原创创作。我也学乐理知识，也学习如何制作如何混编，我也有过能在脑中浮现无限旋律的时候，但是一旦真正打开DAW的时候一切又归于虚无了。这样的状态持续了很久导致我已经不知道为什么创作以及为什么不创作了。 直到某一天读到《被讨厌的勇气》的时候，我注意到一段文中僧人说的话。 “我有一位年轻朋友，虽然梦想着成为小说家，但却总是写不出作品。他说是因为工作太忙、写小说的时间非常有限，所以才写不出来作品，也从未参加过任何比赛。但真是如此吗？实际上，他是想通过不去比赛这一方式来保留一种“如果做的话我也可以”的可能性，即不愿出去被人评价，更不愿去面对因作品拙劣而落选的现实。他只想活在“只要有时间我也可以、只要环境具备我也能写、自己有这种才能”之类的可能性中。或许再过5年或者10年，他又会开始使用“已经不再年轻”或者“也已经有了家庭”之类的借口。” 这让我背脊发凉也似乎让我找到了原因。我在一个自己虚构的位置上坐得太久，反倒不愿意在其他人面前展示了，久而久之刀锈了就更不愿意承认了。 意义 这次Max工程对于我来说不仅是一个课程设计项目，更多的是启发。 给我的最大的启发是对于pattern的认知有了更深入的理解。以前对于创作的理解是在作画中计较每一个像素，在音乐中计较每一帧的表现，这让我在自以为庞大的工作量前心生畏惧。而若是通过pattern的思想来进行创作则会避免这些误区，从心理和实际着手的时候都可以减轻不少负担。 第二是给予了更多创作的信心。这次的系统搭建我也是按照以前仅存的经验一段一段搭建的，仅从和弦来说听不出什么名堂，加上鼓组也是。但是当贝斯和旋律的随机部分出来之后立刻感觉就不一样了。是部分的有机结合成就一首歌，所以在完成各个部分之前听上去不尽人意是情理之中的，如果在这之前就放弃实在可惜。

近况 1 非常艰难的一周过去了。 具体来说一周的时间都在赶DDL，声学和声音编程课的期中作业和项目都放在了这周交，但是从周一开始才开始做。由于完全高估了自己对时间的掌握能力和编辑Latex的能力所以晚上过得特别艰难。 声音编程的项目是完成一个MIDI级别输出的作品，我做了一个随机生成简单Lofi音乐的系统出来。声学的期中是6个高数物理题，花了近18个小时连续从草稿到Latex三百行代码加上Matlab作图完成。 最主要的教训就是不要把事情都放在DDL之前完成。以及更重要的：数学和工程相关的作业不要放在创作之后去完成。因为创作的东西，就比如这一次的声音编程课作业，一旦入迷了就不停修改不停延申不停加功能，前三天半的时间都花在了这上面，导致后面的时间严重不足。如果把声学作业分散开每天完成一些不至于这样狼狈。 2 开学已经快2个月了，感觉状态还是没有调整过来。 一方面是异地他乡还没多少太阳，确实没什么精力。问gpt说可能维生素D不足，得买点保健品吃。 另一方面是对自己的约束似乎不太够，已经离开学校这么久了，重新回到学校也没办法回到高三那种状态，又是一个人，很多时候不太想做就跳过了，导致验收的时候发现没什么成果。 其实现在想想只要lecture预习了总结了，tutorial提前过了，上课再总结了那学习的基本工作就完成了。至于大作业掰碎了做每天都能够过得很充实。 亮点 周记 终于，暄穗开始写周记了。 其实暄穗很早就开始计划写周记了，契机是最早rss关注的一个日本视频博主在博客做教程的同时，每周都会写周记，总结自己的一周。我觉得这能让我抽时间审视自己的一周并总结，同时能够留下记忆，让我回顾的同时也能作为数字骨灰盒留在互联网上。而且想到要把自己的时间进行总结，也就下意识想把生活过得KIRAKIRA一点了。 不过让我真正开始这个行为的契机是这周关注了Blog finder的rss订阅，同时也提交了自己的博客作为收录。这才发现我在互联网上并不是一个孤立的站点，森月台最终能够和其他站台相连接，给城市外的地图增加路线。我看见其他的站长也都积极更新这自己的站点，城市外并不是沉寂的虚无。 Lofi生成系统 这是声音编程课的第一个大作业，也是我最近离艺术最近的一次。 与Rsshub和解 暄穗在和Rsshub斗智斗勇许久之后和它和解了，不再非要按照在云服务器上了。这样也让服务器的其他应用不会卡死了。 暄穗的服务器是一个2g2核的小服务器（实际自己能用的只有1.6g），上面有wordpress，freshrss和一个nginx proxy manager，由于rsshub的存在经常让整个服务器的内存占用在1.3~卡死之间徘徊，也有若干次因为占IOPS和CPU把服务器卡死。所以经过几轮激烈战斗后暄穗还是放弃了，就1.6个G拼啥命啊。 现在暄穗使用的方法是在本地wsl部署一个rsshub(8g的debian够耗的了)，然后通过frp进行内网穿透，这样访问原来域名的时候依旧是rsshub。而且暄穗不在国内，所以暂时某些神奇网站也可以订阅。 之后预计会买一个树莓派把rsshub放上去然后搁寝室一直开着就不用担心断电的问题了。 音乐推荐 MIMI的フローレミ / feat.初音ミク 这是高考后某个下午听见的夏曲，是MIMI的曲风还没有同质化时候我喜欢的旋律，现在听见也能会想起那个什么事情都不用忧虑只需坐在阳台的露营椅专心听歌的时光。 其他 暄穗已经很久没有写过东西了，现在的文字依旧支离破碎，还得适应。我尽量快地赶上来，我想和更多人交流。

托人寄了一份CD的场刊过来，一本书（虽然页数有些少），一个纸袋，几张周边，就当是逛了这一次的同人展了。 持续逛同人展（特指COMIDAY）是从高中开始的，和同学一起在里面逛一天，逛完晚上看电影。高中正是做术力口的狂热时期，逛展的时候也基本上是按着VOCALOID的音乐摊一家一家扫过去的。当时的VOCALOID二创作同人画集的特别多，但是做专辑的数量并不多，见过的就平行四界初心社和稔盛望官方摊位还有一堆寄售其他社团的鸽子，再加上几个海外的社团。 当时逛展子要跑到非遗公园和新城区的会展中心那边基本就是地铁穿个城跑终点站的距离。于是早上五六点钟起床打的坐到火车北站地铁口，坐去那边差不多是早上7点钟。CD不让排夜场，但是为了限定周边7点钟就已经排起人了，于是也一起排着一边背单词做作业等到同学来再等到九点多开场。 逛展前一个晚上就翻完官方的参展动态，把感兴趣的社团都标记一圈，把预算先算出来甚至零钱都准备好准备拿起本子放下钱就跑下一家（当时也没有智能手机，但是基于场地极其糟糕的信号还是零钱来得快），在地图上把最短路线画好。第二天进场安检完速度跑向限定场，然后跑向官方场刊，顺着路线一家一家跑，差不多10分钟就能够把想买的都买到，然后接下来的几个小时就是闲逛时间了。边聊天边看看其他有趣的摊位。 印象深的有一个社团把成都的二次元场地(?)做了一份大的城市地图来卖。一个做术曲（主要是IA和ONE）专辑的海外社团的小哥哥特别可爱，两次去找他签名都会笑着说好多东西（虽然听个半懂）。AVG专区全是严肃otaku不敢去盖印章于是拿痛衣给同学交换代盖。甚至有一届能够看到narcissus的摊位而且还排满人。 每一次逛完回家都会写手记，会写很多东西，灵感泉涌。似乎只有在那样一个看上去不是那么真实的环境下才会想到一些。第一次人生有了非提前制定的目标是因为发现原来自己也可以做到。初次看术曲周刊的时候发现弹幕并不在歌姬而在于P主，这个时候才发现原来一首歌的制作并不是远到明星和企业，就是我这样的普通人也可以完成，于是在这样的契机上开始了尝试。那时资源没有现在这么唾手可得，都藏在各种论坛里面；纸片人也没有像现在这样被发现商机被尝试广传，加上还要补课，于是开始几年都不敢外说，在网友的交流中持续着创作。 尽管知道世界上一定有同好，但是直到站在那里才感受到知道周围这么多人共同爱好着这个领域，在各自的圈子里面进行创作并且乐于向他人展示交流是一种多么激动的感觉。当时的我也决定再次出现在这个地方的时候想要作为创作者与他人交流。 很遗憾，大学之后就没有再参加本地的同人展了。本来有希望的时隔两年的今年暑假也因为成都的本地疫情取消了。不过之后的每一次同人展都会找人带一份场刊过来。现在大陆这方面的纸质杂志基本上所剩无几了，而CD的场刊在这里无疑是质量上乘的（最近见过的其他能够称得上高质量的也就只有只出过三期的AVG SPIRITS了），里面的每一篇约稿采访都能看出执笔者对于这个圈子学术般认真的态度。而且甚至能够在里面找到更早（小学）时见过的展子和游戏馆。不仅带有现实中的老成都情怀，也让我发现更多圈子里上世纪考古人的身影（或许对于他们来说只是想要找寻童年的痕迹吧）。 个人觉得嗅觉应该是最能够唤起人记忆的感觉。虽然封面不同，但开封这份场刊的时候，几年未有过的新印刷杂志特有的油墨味又把我带进了两手拿着纸袋背着书包在场地里奔跑的那几年，似乎又看到了创作者一起交流的场景。很欣慰这份新油墨味确实是新的，证明那里确实有这样的圈子，还有这样的氛围存在着。 (图1.新的场刊，图2.老年机时代唯一找到存下来的CD痕迹，图3.突然想到的@tamuro_g的图片)
