2025.11.30

这个月买了什么 十一月算是购物大月，我虽然不在国内没有办法过双十一，但是还是一贯在黑五买了些东西。 Launchpad Pro mk3 一个19年的老玩意儿了。16年的时候买了Launchpad mk2的版本打了几个灯光效果很酷炫的工程之后就一直用于DrumPad弹和弦，以及和Ableton live一起使用在直播的时候作为通道条的控制器。 在Ebay上面120镑买的几乎全新的mk3，价格和当时闲鱼上面的最低价差不多了。新的版本多了版面的自定义功能可以使用不同的预设。然后多了十分好用的和弦功能和音阶选择功能，很适合没有灵感的时候随机漫游。 以及Pro版本有专属的音序器功能，暂时没有时间去摸索，估计是可以和其他硬件合成器搭配使用的方案。 Mpowersynth 红塔山(Melda production)的波表合成器打骨折，加上折扣券28镑收了，算是逐渐集齐红塔山全家桶的又一步。 特点是极低的cpu消耗以及红塔山的特色效果器路由和无限可能性。 当然还有红塔山特色的让人不想靠近的UI。 Rond 128元 暄穗之前一直使用Life cycle来进行日常的追踪和记录，在使用的过程中发现了其局限性：其一是太长时间没有更新，像是亖了一样。其二是地图如果有所偏移的话修改方式极其诡异甚至无法修改。 所以在少数派搜索Life cycle的时候看见了Rond这款软件。给我的整体感觉就是一个持续更新的Life Cycle，而且还是可以买断的，所以试用了三天就买了。给人感觉相当不错。 同时下载了一个叫做一生足迹的APP一直放在手机后台记录。要说两者的区别的话Rond和Life cycle都是记录事件，记录在一个地方呆的时间有多长。而一生足迹是记录具体地图上经历过的路径。 ELEGOO UNO R3 用于处理学校的Creative Coding for Sound大作业。但是Arduino的价格太高所以选择了替代主板。同时购买的还有ECG贴片，AD8232组件和一盆绿植。大概也能猜出想要做什么东西了。 游戏 亚路塔 关于见习勇者打工打怪建设文明乡镇的故事。一个迷你且完善的类潜水员戴夫的经营动作类游戏。 像素人物很可爱，但是刷钱的过程有些肝弥补了游戏时长短的缺点。估计20小时以内可以玩完。 星之终途 既然是Key社的游戏还是短篇还是4折促销就不得不买了。

2025.10.26

近况 1 非常艰难的一周过去了。 具体来说一周的时间都在赶DDL，声学和声音编程课的期中作业和项目都放在了这周交，但是从周一开始才开始做。由于完全高估了自己对时间的掌握能力和编辑Latex的能力所以晚上过得特别艰难。 声音编程的项目是完成一个MIDI级别输出的作品，我做了一个随机生成简单Lofi音乐的系统出来。声学的期中是6个高数物理题，花了近18个小时连续从草稿到Latex三百行代码加上Matlab作图完成。 最主要的教训就是不要把事情都放在DDL之前完成。以及更重要的：数学和工程相关的作业不要放在创作之后去完成。因为创作的东西，就比如这一次的声音编程课作业，一旦入迷了就不停修改不停延申不停加功能，前三天半的时间都花在了这上面，导致后面的时间严重不足。如果把声学作业分散开每天完成一些不至于这样狼狈。 2 开学已经快2个月了，感觉状态还是没有调整过来。 一方面是异地他乡还没多少太阳，确实没什么精力。问gpt说可能维生素D不足，得买点保健品吃。 另一方面是对自己的约束似乎不太够，已经离开学校这么久了，重新回到学校也没办法回到高三那种状态，又是一个人，很多时候不太想做就跳过了，导致验收的时候发现没什么成果。 其实现在想想只要lecture预习了总结了，tutorial提前过了，上课再总结了那学习的基本工作就完成了。至于大作业掰碎了做每天都能够过得很充实。 亮点 周记 终于，暄穗开始写周记了。 其实暄穗很早就开始计划写周记了，契机是最早rss关注的一个日本视频博主在博客做教程的同时，每周都会写周记，总结自己的一周。我觉得这能让我抽时间审视自己的一周并总结，同时能够留下记忆，让我回顾的同时也能作为数字骨灰盒留在互联网上。而且想到要把自己的时间进行总结，也就下意识想把生活过得KIRAKIRA一点了。 不过让我真正开始这个行为的契机是这周关注了Blog finder的rss订阅，同时也提交了自己的博客作为收录。这才发现我在互联网上并不是一个孤立的站点，森月台最终能够和其他站台相连接，给城市外的地图增加路线。我看见其他的站长也都积极更新这自己的站点，城市外并不是沉寂的虚无。 Lofi生成系统 这是声音编程课的第一个大作业，也是我最近离艺术最近的一次。 与Rsshub和解 暄穗在和Rsshub斗智斗勇许久之后和它和解了，不再非要按照在云服务器上了。这样也让服务器的其他应用不会卡死了。 暄穗的服务器是一个2g2核的小服务器（实际自己能用的只有1.6g），上面有wordpress，freshrss和一个nginx proxy manager，由于rsshub的存在经常让整个服务器的内存占用在1.3~卡死之间徘徊，也有若干次因为占IOPS和CPU把服务器卡死。所以经过几轮激烈战斗后暄穗还是放弃了，就1.6个G拼啥命啊。 现在暄穗使用的方法是在本地wsl部署一个rsshub(8g的debian够耗的了)，然后通过frp进行内网穿透，这样访问原来域名的时候依旧是rsshub。而且暄穗不在国内，所以暂时某些神奇网站也可以订阅。 之后预计会买一个树莓派把rsshub放上去然后搁寝室一直开着就不用担心断电的问题了。 音乐推荐 MIMI的フローレミ / feat.初音ミク 这是高考后某个下午听见的夏曲，是MIMI的曲风还没有同质化时候我喜欢的旋律，现在听见也能会想起那个什么事情都不用忧虑只需坐在阳台的露营椅专心听歌的时光。 其他 暄穗已经很久没有写过东西了，现在的文字依旧支离破碎，还得适应。我尽量快地赶上来，我想和更多人交流。
